## 🐛 Found a bug?

1. Search [existing issues](https://github.com/vdirienzo/neon-serpent/issues)
2. If new, [file a bug report](https://github.com/vdirienzo/neon-serpent/issues/new?template=bug_report.md)
3. Include:
   - Browser + version
   - OS + version
   - Reproduction steps
   - Console errors (if any)

## ✨ Want a feature?

1. Check [open feature requests](https://github.com/vdirienzo/neon-serpent/issues?q=is%3Aissue+label%3Aenhancement)
2. If new, [request a feature](https://github.com/vdirienzo/neon-serpent/issues/new?template=feature_request.md)
3. Consider opening a PR — we love contributions!

## 🛡️ Security issue?
